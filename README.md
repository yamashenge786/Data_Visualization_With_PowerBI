# 📊 Data Professional Survey Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive **Power BI dashboard** that visualizes survey data collected from data professionals.  

The dashboard provides clear insights into:

- 💰 Average salary by gender and job title  
- 💻 Most popular programming languages  
- 😊 Job satisfaction metrics (work-life balance & salary happiness)  
- 📈 Survey participation statistics  

This project demonstrates strong data visualization, business intelligence, and analytical storytelling skills.

---

## 🎯 Business Problem

Recruiters, HR teams, and hiring managers often lack clear, data-driven insights into:

- Salary benchmarks across different roles and genders  
- Trending programming languages in the data industry  
- Employee satisfaction levels  
- Workforce compensation expectations  

Without these insights, compensation planning and hiring strategies become guesswork.

---

## ✅ Solution

This dashboard solves the problem by transforming raw survey data into:

- Interactive visual insights  
- Easy-to-interpret KPIs  
- Comparative salary analysis  
- Industry trend breakdowns  

The dashboard supports **better hiring decisions, workforce planning, and compensation strategy**.

---

## 🛠️ Tools & Technologies Used

- **Power BI** – Data visualization & dashboard development  
- **Power BI Data Modeling** – Relationships, measures, calculated columns  
- **Survey Dataset (CSV/Excel)** – Data source  
- DAX (for calculated metrics and KPIs)

---

## 📊 Key Features

- 🔎 Interactive slicers (filter by job title, gender, etc.)
- 📈 KPI cards showing:
  - Average salary
  - Total survey participants
- 📊 Salary comparison visuals by job title and gender
- 💻 Programming language popularity breakdown
- 🎯 Gauge visuals displaying:
  - Work-life balance satisfaction
  - Salary happiness score
